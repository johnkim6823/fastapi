# fastapi
## download requirement
```
pip3 install fastapi uvicorn httpx
```

## go to fastapi folder
```
cd fastapi
```

## activate fastapi
```
uvicorn server:app --reload --port 5000
```
```
uvicorn [name]:[app_name] --reload --port [port]
```
